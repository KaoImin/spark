# Run
```
cargo run -- -i  // init tx

cargo run -- -m  // mint tx

// stake tx
cargo run -- -s first   // first stake
cargo run -- -s add     // add stake
cargo run -- -s redeem  // redeem stake

// delegate tx
cargo run -- -s first   // first delegate
cargo run -- -s add     // add delegate
cargo run -- -s redeem  // redeem delegate
```
